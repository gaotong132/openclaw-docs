## 一、项目概述

### 1.1 产品定位

| 项目 | 描述 |
|------|------|
| **名称** | 班级宠物园 (ClassPetGarden) |
| **定位** | 班级积分管理 + 电子宠物养成系统 |
| **用户** | 教师、学生 |
| **核心价值** | 游戏化班级管理，提升学生积极性 |
| **特点** | 终生免费、微信集成 |

## 二、完整功能清单

### 2.1 顶部导航栏

| 功能 | 说明 |
|------|------|
| **班级选择器** | 下拉切换班级，显示"班级宠物园" + 当前班级名 |
| **设置与帮助** | 应用设置和使用帮助入口 |
| **下载到桌面** | PWA 安装 / 桌面快捷方式 |

### 2.2 左侧工具栏

| 功能 | 说明 |
|------|------|
| **搜索学生** | 搜索框，快速查找学生 |
| **姓名排序** | 按姓名排序学生列表（升序/降序切换） |
| **排行榜** | 荣誉榜：排名、学生、等级、徽章数 |
| **小商店** | 积分兑换商品，库存管理 |
| **评价记录** | 历史评价记录，支持导出Excel |
| **撤回评价** | 撤销最近一次评价操作 |
| **批量评价** | 选择多个学生批量加分/扣分 |

### 2.3 班级管理

| 功能 | 说明 |
|------|------|
| **创建班级** | 输入班级名称创建新班级 |
| **编辑班级** | 修改班级名称 |
| **删除班级** | 删除班级及学生数据 |
| **切换班级** | 顶部下拉选择切换 |
| **当前使用** | 显示当前使用的班级 |

### 2.4 学生管理

| 功能 | 说明 |
|------|------|
| **添加学生** | 输入姓名 + 学号（可选） |
| **批量导入** | 批量导入学生名单 |
| **编辑学生** | 修改姓名、学号 |
| **删除学生** | 删除学生及宠物数据 |
| **搜索学生** | 按姓名搜索 |
| **姓名排序** | 按姓名排序 |

### 2.5 宠物系统

#### 宠物选择（24种）

**普通动物（16种）**：
| 类型 | 名称 |
|------|------|
| 犬类 | 西高地、比熊、边牧、柴犬、金毛、萨摩耶、哈士奇 |
| 猫类 | 虎斑猫、加菲猫、布偶猫、橘猫 |
| 兔类 | 垂耳兔、安哥拉兔 |
| 其他 | 柯尔鸭、羊驼、小熊猫 |

**神兽类（7种）**：
| 类型 | 名称 |
|------|------|
| 神兽 | 白虎、独角兽、貔貅、青龙、朱雀、狻猊 |
| 精灵 | 多肉精灵 |

#### 宠物属性

| 属性 | 说明 |
|------|------|
| **等级** | Lv.1 ~ Lv.8 |
| **成长值** | 当前等级的积分进度 |
| **更换宠物** | 可随时更换宠物类型 |

#### 成长设置（等级配置）

| 升级 | 所需积分 |
|------|----------|
| Lv.1 → Lv.2 | 40 |
| Lv.2 → Lv.3 | 60 |
| Lv.3 → Lv.4 | 80 |
| Lv.4 → Lv.5 | 100 |
| Lv.5 → Lv.6 | 120 |
| Lv.6 → Lv.7 | 140 |
| Lv.7 → Lv.8 | 160 |
| **Lv.8** | 毕业，获得徽章 |

#### 徽章系统

- 宠物达到满级（Lv.8）后完成毕业仪式获得专属徽章
- 徽章数量代表养成宠物数量
- 荣誉榜显示徽章数

## 四、数据模型

```typescript
// 班级
interface Class {
  id: string;
  name: string;
  teacherId: string;
  students: Student[];
  createdAt: number;
}

// 学生
interface Student {
  id: string;
  name: string;
  studentNo?: string;       // 学号
  classId: string;
  pet?: Pet;
  createdAt: number;
}

// 宠物
interface Pet {
  id: string;
  type: string;             // 宠物类型（西高地、白虎等）
  level: number;            // 等级 1-8
  exp: number;              // 当前成长值
  adoptedAt: number;
}

// 评价记录
interface EvaluationRecord {
  id: string;
  classId: string;
  studentId: string;
  studentName: string;
  points: number;           // 正数加分，负数扣分
  reason: string;
  category: '学习' | '行为' | '健康' | '其他';
  timestamp: number;
}

// 商店商品
interface ShopItem {
  id: string;
  name: string;
  description: string;
  price: number;            // 积分价格
  stock: number;            // 库存
  image?: string;
}

// 兑换记录
interface ExchangeRecord {
  id: string;
  studentId: string;
  itemId: string;
  itemName: string;
  price: number;
  timestamp: number;
}

// 徽章
interface Badge {
  id: string;
  studentId: string;
  petType: string;          // 毕业宠物类型
  earnedAt: number;
}

// 成长配置
interface LevelConfig {
  level: number;            // 目标等级
  requiredExp: number;      // 所需积分
}
```
